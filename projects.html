TolkCare App

Android · Kotlin · Jetpack Compose · Firebase · Firestore · FCM · Agora

TolkCare er en Android-app til booking af tolke. Brugere kan finde tolke, vælge dato og tidspunkt, se deres bookinger og aflyse igen. Tolken kan registrere arbejdstider, se bookinger og få notifikationer, når nye bookinger oprettes.

  • UI bygget med Jetpack Compose.
  • Firebase Authentication til login og brugerhåndtering.
  • Firestore til lagring af brugere, tolke, arbejdstider og bookinger.
  • Push-notifikationer med FCM til tolken ved nye bookinger.
  • Voice/video-calls mellem bruger og tolk via Agora SDK.

IoT Waste Management

MQTT · EMQX · .NET · Firestore · IoT

IoT Waste Management er et smart affaldssystem, som overvåger fyldningsniveau i skraldespande. Sensor-data sendes via MQTT til en EMQX-broker og videre til backend, hvor data gemmes og bruges til at optimere tømmeruter.

  • Fake datagenerator der sender afstand og GPS-data via MQTT.
  • EMQX broker (Docker) til publish/subscribe af sensordata.
  • .NET backend der modtager og gemmer data.
  • Visualisering af fyldningsniveauer og alarmer.

Model Management System – React Web App

React · JavaScript · JWT Login · REST API · Docker Backend

Dette projekt er en React-applikation udviklet til et modelbureau, hvor medarbejdere kan administrere modeller, managere og jobopgaver. Systemet bruger en Docker-hostet backend, JWT-login og et komplet REST API.

  • Login med JWT og automatisk token-håndtering.
  • Roller: Manager og Model med forskellige rettigheder.
  • Manager kan oprette modeller, managere, jobs og tilføje modeller til jobs.
  • Model kan se sine jobs og registrere udgifter.
  • React SPA med dynamisk UI baseret på brugerens rolle.
  • Integration til Docker backend og REST endpoints.
