TolkCare App

Android · Kotlin · Jetpack Compose · Firebase · Firestore · FCM · Agora (voice/video)

TolkCare er en Android-app til booking af tolke. Brugeren kan finde en tolk, vælge tidspunkt, se egne bookinger og kommunikere direkte med tolken via voice/video. Appen er bygget som et komplet system med både bruger- og tolkeflows.

Mit ansvar
  • Design og implementering af UI i Jetpack Compose.
  • Firebase Authentication til login og oprettelse af brugere.
  • Firestore-datamodel til brugere, tolke, arbejdstider og bookinger.
  • Notifikationer med FCM (tolk får besked, når en bruger laver en booking).
  • Integrering af Agora SDK til voice/video-samtaler mellem bruger og tolk.
  • Håndtering af logik til booking, sletning, favorit-tolke og kalender-visning.
Tekniske pointer
  • MVVM-arkitektur med ViewModels og repository-lag.
  • Reaktiv UI med Jetpack Compose og state-håndtering.
  • Cloud Firestore-dokumentstruktur til bookings, tolk-profiler og arbejdstider.
  • Brug af FCM HTTP API til målrettede notifikationer til specifikke tolke-enheder.

IoT Waste Management System

IoT · Raspberry Pi · MQTT · EMQX · .NET backend · Android/visualisering · Firestore

IoT Waste Management System er et smart affaldssystem, der overvåger fyldningsniveauet i skraldespande. Tanken er at optimere tømmeruter, så skraldespande kun tømmes, når det er nødvendigt. Systemet kombinerer sensordata, MQTT-kommunikation, backend og app-visning.

Mit ansvar
  • Opsætning af MQTT-broker med EMQX (Docker).
  • Design af datamodel for waste bins (id, placering, fyldningsniveau, alert-status).
  • Udvikling af .NET backend til at modtage, gemme og bearbejde sensor-data.
  • Android-visning / dashboard til at se fyldningsniveauer og alarmer.
  • Implementering af fake datagenerator til at simulere sensordata.
Tekniske pointer
  • MQTT topics til hver skraldespand (bin) og håndtering af real-time data.
  • EMQX som broker, hostet via Docker.
  • .NET API der persisterer data (fx i Firestore/DB) og eksponerer endpoints.
  • Logik til at sætte alarmer, når en skraldespand er over en vis fyldningsprocent.
